## DMIT 是什么：一家认真做网络的华人 VPS 商

DMIT 2018 年开始运营，纽约注册的正规公司（编号 5246271），背后是华人团队，有中文客服渠道，支持支付宝、微信、PayPal 付款——这对国内用户来说算是很友好的基础条件了。

硬件层面没什么特别的——全系 AMD EPYC 处理器、KVM 虚拟化、企业级 SSD，这些在高端 VPS 里算标配。DMIT 真正不一样的地方在网络：他们自建机房、自有带宽，和中国三大运营商直接签约。手里握着 CN2 GIA 40Gbps、AS9929 20Gbps、CMIN2 20Gbps 的资源，这是真金白银砸进去的基础设施，不是转卖套壳。

有一个细节能说明 DMIT 的定位：搬瓦工洛杉矶 DC6 机房的 CN2 GIA-E 带宽，就是从 DMIT 采购的。它家是上游，不是转卖商。

**什么是 CN2 GIA？**

CN2 GIA（China Telecom Next Carrier Network，Global Internet Access）是中国电信旗下专为国际流量优化的骨干网，区别于普通 163 骨干网的是，CN2 GIA 采用独立路由，拥挤时不降级、不绕路。简单说：别的线路晚高峰掉速，CN2 GIA 通常稳得住。

## 四个机房的特点

**洛杉矶（LAX）**：DMIT 的主战场，产品线最全。Pro 系列电信联通去程 CN2 GIA、移动走 CMI，三网回程统一走 CN2 GIA，实测非高峰延迟在 155-165ms 区间，丢包率极低。EB 系列（Eyeball）则用三网 CMIN2 回程，联通移动用户用起来很顺手，价格比 Pro 便宜一档。另外还有两个特殊系列：LAX.sPro 带 5Tbps+ DDoS 防护，适合游戏服务器或容易被打的场景；LAX.Pro.u 是不限量版本，视频站、下载站不用担心超量。

**圣何塞（SJC）**：T1 国际线路 + 自带 20Gbps DDoS 防护，是 DMIT 里最便宜的入门选项，年付 $36.9 起。没有中国大陆优化，但防护到位，适合跑国际业务或者纯建站。流量超限不断网，降速到 50Mbps 继续用，这个设计挺实在的。

**香港（HKG）**：延迟最低，国内访问通常在 20-50ms 之间，是大陆用户体验最直接的机房。Premium 系列走电信 CN2 GIA + 联通 AS9929 + 移动 CMI，延迟低、稳定性好。代价是价格偏高，TINY 套餐月付接近 $40，不适合纯粹省钱的场景。Eyeball 系列价格低一档，CMI 双向，移动用户首选。

**东京（TYO）**：地理位置介于香港和洛杉矶之间，延迟大概在 50-80ms。Pro 系列和香港 Premium 一样的线路配置，日本 IP 在某些场景有天然优势，比如需要日本本地资源或者 TikTok 相关业务。注意：东京 Eyeball 系列已经下架了，现在只有 Pro 和 T1 可选。

## 怎么用优惠码：三步搞定

1. 进入购物车页面，选好套餐和付款周期
2. 找到 **"Promo Code"** 输入框，复制粘贴对应优惠码（建议直接复制，手打容易出错）
3. 点击 **"Validate Code"**，页面价格实时更新后确认折扣，再提交订单

几个注意点：每账户每次只能用一个优惠码；循环折扣意味着每次续费都按折扣价，不存在"首年优惠坑"；大多数优惠码要求**季付或更长付款周期**才能激活。

## 讲真，DMIT 有什么缺点

不超售这件事，有好有坏。好处是你买到的带宽是真实的，晚高峰不会因为隔壁用户跑满带宽影响你。坏处是套餐经常缺货——热门机型，比如香港 Pro 的 VICTORIA，补货后几小时就能卖光。如果你看中某个套餐有货，真的不要拖。

工单客服有中文渠道，但回复是英文，而且响应周期大概 72 小时，不是"遇到问题马上有人处理"的节奏。如果你有紧急运维需求，这点要提前知道。

价格确实不便宜。香港 Pro TINY 月付接近 $40，对比那些几块钱的"香港服务器"，差距很明显。但要类比的话，DMIT 卖的是线路质量，不是机器资源本身。
